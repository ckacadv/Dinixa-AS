www.renatogr.cf Desktop 4.5 is out! April 23rd 2023 After almost 3 months from the scheduled release date rgd 4.5. It's finally here, the new version brings a new UI and new features. The home page has been edited so that everything is now one below the other. A new widget has been added that can show a picture or gif. The navigation bar has been changed, now it is on the side. In that nav bar there can also be some sort of list, for example on the testers tab the nav bar is also used to manage the page. There are also minor updates on other tabs.
